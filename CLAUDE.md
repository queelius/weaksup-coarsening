## Project Overview

Academic paper repository: **Programmatic weak supervision as
masked-cause inference: identifiability of label models without gold
data.**

Conference-format target (~12 pages). Fourth paper in a series that
ports the masked-data identifiability framework from reliability
statistics to new application domains, after series-system
reliability, scRNA-seq zero inflation, and spatial-transcriptomics
deconvolution.

## Build Commands

```bash
make paper      # builds main.pdf
make sim        # runs scripts/run.R, writes results.rds
make figures    # runs scripts/figures.R
make clean      # removes LaTeX artifacts
```

## Architecture

- `main.tex`: pure-LaTeX top-level with preamble + `\input{sections/...}`
- `sections/`: 8 section files (no `\end{document}` in section files)
- `refs.bib`: BibTeX
- `figures/`: PDF figures generated by `scripts/figures.R`
- `scripts/`: base-R simulation
  - `sim.R`: DGP, estimators, diagnostics (sourced by run.R)
  - `run.R`: four studies (one per theorem), saves `results.rds`
  - `figures.R`: reads `results.rds`, writes the two figures

## The four theorems

- T1 glass ceiling (`identifiability.tex`): gold-free
  non-identifiability via the accuracy-complement symmetry.
- T2 conditional-independence identifiability (`identifiability.tex`):
  re-derives Dawid--Skene / data-programming / triplet identifiability
  as a masked-cause theorem.
- T3 agreement consistency (`identifiability.tex`): the label-model
  analogue of cell-total consistency.
- T4 gold-set sample complexity (`methodology.tex`): the genuinely new
  result, `O(r / gap^2)` gold examples for the dependent case.

## Prior-art honesty (critical)

Label-model identifiability is established prior work. Dawid--Skene
(1979) is the clear ancestor; data programming (Ratner 2016) and
FlyingSquid (Fu 2020) proved moment-based identifiability for
conditionally independent LFs. Do NOT claim to invent label-model
identifiability. The contribution is the masked-cause unification,
the C1--C2--C3 classification, the explicit glass-ceiling
construction, and the gold-set sample-complexity theorem (T4) for the
dependent case. `discussion.tex` positions this honestly.

## Conference format constraints

statements; two essential figures; tables over lengthy text.
